# AI-Based Fruit Recognition System

This project is a computer vision application that recognizes fruits using a webcam and deep learning.

## Features
- Real-time camera capture
- Image classification using Hugging Face
- Fruit prediction with confidence score
- Simple and interactive interface

## Technologies Used
- Python
- OpenCV
- Hugging Face Transformers
- Deep Learning
- Computer Vision

## How It Works
1. The webcam captures a fruit image.
2. The image is processed using a pretrained Hugging Face model.
3. The system predicts the fruit category with a confidence score.

## Installation

Install dependencies:

pip install transformers torch pillow opencv-python

Run the project:

python app.py
